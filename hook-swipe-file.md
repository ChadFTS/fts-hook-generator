## üéÅ BONUS: PLATFORM-SPECIFIC TIPS

### LinkedIn
- Start with a one-liner hook, then line break
- Use "I" statements ‚Äî personal stories win
- End with a question to drive comments

### YouTube
- Put the hook in the first 5 seconds
- Use pattern interrupts (visual changes)
- Promise the payoff clearly

### TikTok/Reels
- Hook in first 1-2 seconds or they scroll
- Text on screen reinforces verbal hook
- Controversy and curiosity = views

### Twitter/X
- First line IS the hook ‚Äî no wasted words
- Threads: Each tweet should stand alone
- Quote tweets with hot takes perform well
